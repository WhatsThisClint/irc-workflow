# Inception Report Review — Output Template

Use this section order for the downloadable Word document (via the docx skill). Keep language direct and specific — every claim should point to something in the source report, not a generic template phrase.

1. **Cover summary** (report title/researcher name if known, discipline reviewed under, one-paragraph overall verdict)
2. **Scientific Rigor Audit** — table or subheadings for Objectivity, Validity, Reliability, Transparency, Replicability, each with verdict + evidence + gap
3. **Conceptual Model** — extracted model (relationship map), variable classification, justification check
4. **Literature Review Assessment** — mode (summary vs argument), gap statement quality, engagement with opposing frameworks, citation notes
5. **Coherence & Anticipated Critique** — assumptions, counterfactuals, likely reviewer pushback and whether addressed
6. **Disciplinary Lens Notes** — lens(es) applied, discipline-specific concerns
7. **Summary & Recommendations** — peer-review readiness verdict, ranked top gaps, concrete fix per gap (named readings/frameworks/design changes)

Keep the doc scannable: use headers, short paragraphs, and bullet lists for gap/fix pairs rather than dense prose blocks.
