# Work Plan Construction

Produce a month-by-month table of activities and milestones, sized to the stated or assumed project duration (default 6-8 months if not specified — state the assumption).

## Standard phase skeleton (adapt names/count of months to the specific study)
1. **Literature review & tool design** — finalize conceptual model, draft survey/data-extraction instrument, identify sampling frame.
2. **Piloting** — pilot the instrument/method on a small subsample, revise based on pilot findings.
3. **Data collection / fieldwork** — main survey, interviews, or remote-sensing data pull and ground-truthing, sized to sample size from the study design.
4. **Data cleaning & analysis** — coding (qualitative) or statistical/spatial analysis (quantitative).
5. **Draft report & internal review**.
6. **Final report & dissemination to client**.

## Output form
A markdown table: rows = activities/milestones, columns = months (numbered, e.g. 1 through however many), with an X marking active months — mirroring the Gantt-style format WELL Labs' own inception report template uses. Keep activity names specific to this study, not generic placeholders (e.g. "Pilot WUC interview guide with 3 WUCs in Raichur" rather than "Pilot testing").
