# Instrument Drafting

Draft a short annex instrument — every item must map to a named variable from the conceptual model. Don't pad with generic demographic questions unless a variable actually needs them (e.g. household size only matters if it's a named control).

## For interview/survey-based studies
- Group items by the variable they measure (IV items, DV items, mediator items, moderator items, key controls).
- Use a mix of closed (for variables that need comparison across respondents) and open items (for mechanism/mediator questions where "why" matters).
- Keep it short — 10-20 items for an inception-stage draft, not a full field-ready instrument; note that it should be piloted and refined per the work plan.
- Label each item or group with the variable it targets in brackets, e.g. "[DV: fund transparency] How often are WUC financial statements shared with members?"

## For remote-sensing/quantitative studies
Replace the survey with a **data-extraction protocol**: list the specific data layers/fields to be extracted (e.g. NDVI time series per plot, irrigation source label, crop type, sowing date), the source for each, and the ground-truthing checklist items for the subsample used in validation.

## Always
State plainly that this is a draft for piloting, not a final instrument — consistent with the work plan's piloting phase.
