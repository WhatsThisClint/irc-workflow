# Literature Review Construction — Argument Mode

Build the framing so it would already pass `inception-report-review`'s summary-vs-argument test — never write a list of "Smith found X, Jones found Y."

## What to produce
1. **Name the dominant existing framework(s)** for this kind of question in the identified discipline (e.g. for cess/cost-recovery: theories of collective action and common-pool resource governance, e.g. the Ostrom design-principles tradition; for remote-sensing water-source attribution: vegetation-index-based crop water stress literature).
2. **State a specific gap** this study fills — not "more research is needed," but something concrete: "most WUC performance studies compare funding modalities across states without controlling for pre-existing institutional capacity; this study's paired-comparison design addresses that," or "prior NDVI-based irrigation-source studies rely on single-date imagery; this study uses a time-series comparison across the crop cycle to distinguish canal vs. well signatures more reliably."
3. **Name real anchor papers/frameworks only if you actually know them** — well-known, citable work (e.g. Ostrom on commons governance, established remote-sensing water-stress indices). If you're not confident a specific citation is real and correctly attributed, describe the *type* of source that should anchor this section (e.g. "peer-reviewed literature on canal-vs-groundwater NDVI discrimination in semi-arid India") rather than inventing an author/year. Flag clearly in the output which citations are illustrative placeholders for the researcher to verify/replace with a proper search, versus ones you're confident are real.
4. **Engage a competing explanation**, not just the supportive framework — e.g. an alternative account of why compliance differs across states (state capacity/enforcement differences) versus the institutional-design account the study centers on. Note this competing account explicitly so the study design can address or bracket it.
5. **Separate theoretical justification from feasibility justification** for the chosen method — state both, but don't let feasibility reasons stand in for a theoretical one when a major design choice is being justified.

## Output form
2-4 short paragraphs, not a long academic review — this is an inception report, not a thesis. Argument-mode, gap-stated, one competing framework acknowledged, citations flagged as verified/illustrative per point 3 above.
